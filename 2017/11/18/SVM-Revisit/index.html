SVM Revisit

SVM (Support Vector Machine) is an important tool in pattern recognition before the wave of different kind of neural networks. In this post, we revisit the derivation of SVM and get some important intuition of the result.

The original problem

Suppose we have some training data D={(x1,y1),(x2,y2),⋯ ,(xn,yn)}\mathcal{D}=\{(\mathbf{x}_1, y_1),(\mathbf{x}_2, y_2),\cdots,(\mathbf{x}_n, y_n)\}D={(x1​,y1​),(x2​,y2​),⋯,(xn​,yn​)}, in which x\mathbf{x}x is a feature vector, and yiy_iyi​ is the class label for each sample and it is -1 or 1. Our task is to find a classifier that uses a separate hyper plane to divide the points in D\mathcal{D}D into two parts. Points on each side of the hyper plane have the same class label y respectively. This classifier is denoted as f(x)=sign(wTx+b)f(\mathbf{x})=\text{sign}(\mathbf{w}^T\mathbf{x}+b)f(x)=sign(wTx+b). For any hyper plane that successfully divide the two classes. We define the geometry margin of the iiith point as the distance from the point to the hyper plane:

γi=yi(wTxi+b)∣∣w∣∣\gamma_i=\frac{y_i(\mathbf{w}^Tx_i+b)}{||\mathbf{w}||} γi​=∣∣w∣∣yi​(wTxi​+b)​


We observe that for both classes y=1y=1y=1 and y=−1y=-1y=−1, the geometry margin is larger than 0. We can define the geometry margin of our dataset D\mathcal{D}D as:

γ=min⁡i=1,⋯ ,nγi\gamma=\min_{i=1,\cdots,n} \gamma_i γ=i=1,⋯,nmin​γi​

We can see that there are a lot of possible hyper planes. Our ultimate goal is to find the best hyper plane (with parameter (w∗,b∗)(\mathbf{w}^*,b^*)(w∗,b∗)) that maximize the geometry margin of the dataset D\mathcal{D}D. Only the geometry margin of the dataset is maximized, can some points between two classes been classified correctly. We can write our problem in the standard optimization form as:

(P)               max⁡w,bγs.t. yi(wTxi+b)∣∣w∣∣≥γ,i∈[1,n]\begin{aligned} (P)\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ &\max_{\mathbf{w},b} \gamma\\ s.t.\ & \frac{y_i(\mathbf{w}^T\mathbf{x_i}+b)}{||\mathbf{w}||} \ge \gamma,i \in [1, n] \end{aligned}(P)               s.t. ​w,bmax​γ∣∣w∣∣yi​(wTxi​+b)​≥γ,i∈[1,n]​

Convert the original problem into its dual form

To make our problem easy to solve, we can convert it into its dual form. First of all, we define the function margin to help our further derivations. A function margin of iiith point is defined as:

γi^=yi(wTx+b)\hat{\gamma_i}=y_i(\mathbf{w}^T\mathbf{x}+b) γi​^​=yi​(wTx+b)

The function margin of the whole dataset is defined as:

γ^=min⁡i=1,⋯ ,nγ^i\hat\gamma=\min_{i=1,\cdots,n} \hat\gamma_i γ^​=i=1,⋯,nmin​γ^​i​

It can be easily observed that γ=γ^∣∣w∣∣\gamma=\frac{\hat\gamma}{||\mathbf{w}||}γ=∣∣w∣∣γ^​​.
The original problem can be rewritten as:

max⁡w,bγ^∣∣w∣∣s.t. yi(wTxi+b)≥γ^,i∈[1,n]\begin{aligned} &\max_{w,b} \frac{\hat\gamma}{||\mathbf{w}||}\\ s.t.\ & y_i(\mathbf{w}^T\mathbf{x_i}+b) \ge \hat\gamma,i \in [1, n] \end{aligned}s.t. ​w,bmax​∣∣w∣∣γ^​​yi​(wTxi​+b)≥γ^​,i∈[1,n]​

We observed that γ^\hat\gammaγ^​ can be assigned to any value, because if we change γ^\hat\gammaγ^​ to λγ^\lambda\hat\gammaλγ^​, we can adjust (www, bbb) to (λw\lambda \mathbf{w}λw, λb\lambda bλb) to make the problem unchanged. As a result, we can assign γ^=1\hat\gamma=1γ^​=1 to simplify the problem, i.e.

max⁡w,b1∣∣w∣∣s.t. yi(wTxi+b)≥1,i∈[1,n]\begin{aligned} &\max_{\mathbf{w},b} \frac{1}{||\mathbf{w}||}\\ s.t.\ & y_i(\mathbf{w}^T\mathbf{x_i}+b) \ge 1,i \in [1, n] \end{aligned}s.t. ​w,bmax​∣∣w∣∣1​yi​(wTxi​+b)≥1,i∈[1,n]​

To maximize $ \frac{1}{||\mathbf{w}||}$ is the same as minimize ∣∣w∣∣||\mathbf{w}||∣∣w∣∣, and the problem changes to:

min⁡w,b12∣∣w∣∣2s.t. yi(wTxi+b)≥1,i∈[1,n]\begin{aligned} &\min_{\mathbf{w},b} \frac{1}{2}||\mathbf{w}||^2\\ s.t.\ & y_i(\mathbf{w}^T\mathbf{x_i}+b) \ge 1,i \in [1, n] \end{aligned}s.t. ​w,bmin​21​∣∣w∣∣2yi​(wTxi​+b)≥1,i∈[1,n]​

Next, we want to merge the constrains into the objective function to make it an unconstrained problem. We want to design a function gi(w,b)g_i(\mathbf{w},b)gi​(w,b), s.t.

gi(w,b)={0   if (w,b) fulfill the ith constrain∞   otherwiseg_i(\mathbf{w},b)=\left\{ \begin{aligned} 0&\ \ \ \text{if } (\mathbf{w},b) \text{ fulfill the ith constrain}\\ \infty&\ \ \ \text{otherwise} \end{aligned} \right.gi​(w,b)={0∞​   if (w,b) fulfill the ith constrain   otherwise​

It can be written into a more compact form with the max⁡\maxmax notation:

gi(w,b)=max⁡αi≥0αi[1−yi(wTx+b)]g_i(\mathbf{w},b)=\max_{\alpha_i\ge0}\alpha_i[1-y_i(\mathbf{w}^T\mathbf{x}+b)] gi​(w,b)=αi​≥0max​αi​[1−yi​(wTx+b)]

In which, αi\alpha_iαi​ is called the Largrange multiplier. If (w,b)(\mathbf{w},b)(w,b) fulfill the constrain, 1−yi(wTx+b)≤01-y_i(\mathbf{w}^T\mathbf{x}+b)\le01−yi​(wTx+b)≤0 and αi=0\alpha_i=0αi​=0 to attain its max value 0. On the other hand, if (w,b)(\mathbf{w},b)(w,b) violate the constraint, 1−yi(wTx+b)>01-y_i(\mathbf{w}^T\mathbf{x}+b)>01−yi​(wTx+b)>0 and αi=∞\alpha_i=\inftyαi​=∞ to attain its max value ∞\infty∞.
Thus, the optimization problem can be written as $$\begin{aligned}&\min_{\mathbf{w},b}\frac{1}{2}||\mathbf{w}||2+\sum_{i=1}ng_i(\mathbf{w},b)\
=&\min_{\mathbf{w},b}\frac{1}{2}||\mathbf{w}||2+\sum_{i=1}n\max_{\alpha_i\ge0}\alpha_i[1-y_i(\mathbf{w}^T\mathbf{x}+b)]\
=&\max_{\alpha_i\ge0}\min_{\mathbf{w},b}\frac{1}{2}||\mathbf{w}||2+\sum_{i=1}n\alpha_i[1-y_i(\mathbf{w}^T\mathbf{x}+b)]\
=&\max_{\alpha_i\ge0}\min_{\mathbf{w},b}J(\mathbf{w},b,\mathbf{\alpha})
\end{aligned}$$
In which J(w,b,α)=12∣∣w∣∣2+∑i=1nαi[1−yi(wTx+b)]J(\mathbf{w},b,\mathbf{\alpha})=\frac{1}{2}||\mathbf{w}||^2+\sum_{i=1}^n\alpha_i[1-y_i(\mathbf{w}^T\mathbf{x}+b)]J(w,b,α)=21​∣∣w∣∣2+∑i=1n​αi​[1−yi​(wTx+b)]. The inside minimize problem can be solved by setting the derivative to 0:

∂J(w,b,α)∂w=w−∑i=1nαiyixi=0∂J(w,b,α)∂b=−∑i=1nαiyi=0\begin{aligned} \frac{\partial J(\mathbf{w},b,\mathbf{\alpha})}{\partial \mathbf{w}}&=\mathbf{w}-\sum_{i=1}^n\alpha_iy_i\mathbf{x_i}=0\\ \frac{\partial J(\mathbf{w},b,\mathbf{\alpha})}{\partial \mathbf{b}}&=-\sum_{i=1}^n\alpha_iy_i=0 \end{aligned}∂w∂J(w,b,α)​∂b∂J(w,b,α)​​=w−i=1∑n​αi​yi​xi​=0=−i=1∑n​αi​yi​=0​

We get w∗=∑i=1nαiyixi\mathbf{w^* }=\sum_{i=1}^n\alpha_iy_i\mathbf{x_i}w∗=∑i=1n​αi​yi​xi​, substitute it into the optimization problem we get:

max⁡αi≥0∑i=1nαi−12∑i=1n∑j=1nyiyjαiαjxiTxj\max_{\alpha_i\ge0}\sum_{i=1}^n\alpha_i-\frac{1}{2}\sum_{i=1}^n\sum_{j=1}^ny_iy_j\alpha_i\alpha_j\mathbf{x_i}^T\mathbf{x_j} αi​≥0max​i=1∑n​αi​−21​i=1∑n​j=1∑n​yi​yj​αi​αj​xi​Txj​

We write it into a formal optimization problem:

(D)               min⁡α  12∑i=1n∑j=1nyiyjαiαjxiTxj−∑i=1nαis.t.  ∑i=1nαiyi=0αi≥0,i∈[1,n]\begin{aligned} (D)\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \min_{\alpha}\ \ & \frac{1}{2}\sum_{i=1}^n\sum_{j=1}^ny_iy_j\alpha_i\alpha_j\mathbf{x_i}^T\mathbf{x_j}-\sum_{i=1}^n\alpha_i \\ s.t. \ \ &\sum_{i=1}^n\alpha_iy_i=0\\ &\alpha_i\ge0, i \in [1,n] \end{aligned}(D)               αmin​  s.t.  ​21​i=1∑n​j=1∑n​yi​yj​αi​αj​xi​Txj​−i=1∑n​αi​i=1∑n​αi​yi​=0αi​≥0,i∈[1,n]​

This is the dual optimization problem of problem (P) and it is a standard Quadratic Programming which can be solved by a lot of solvers.

Some obervations and intuitions

  1. The variable of the primal optimization problem (P) is to optimize (w,b)(\mathbf{w},b)(w,b) while the dual problem (D) is to optimize the dual variable α\mathbf{\alpha}α. The length of dual variable equals to the number of constrains of the primal problem.

  2. If αi=0\alpha_i=0αi​=0, yi(wTxi+b)>1y_i(\mathbf{w}^T\mathbf{x_i}+b)>1yi​(wTxi​+b)>1 the function margin of this is not equal to the function margin of the dataset (γ^=1\hat\gamma=1γ^​=1) . It means that the margin of this sample is not the smallest among others.
    If αi>0\alpha_i>0αi​>0, it means that the iiith constrain is active and yi(wTxi+b)=1y_i(\mathbf{w}^T\mathbf{x_i}+b)=1yi​(wTxi​+b)=1. The function margin of the iiith sample equals to the function margin of the dataset. This sample is called a support vector.

  3. We observe the optimal value of w∗\mathbf{w^*}w∗:

w∗=∑i=1nαiyix\mathbf{w}^*=\sum_{i=1}^n\alpha_iy_i\mathbf{x} w∗=i=1∑n​αi​yi​x

In which αi=0\alpha_i=0αi​=0 for all non support vectors, which means that the value of w∗\mathbf{w^*}w∗ is only determined by support vectors.
